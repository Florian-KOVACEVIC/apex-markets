# Compass Pro — Analyse Financiere

Application d'analyse financiere avancee avec scoring quantitatif et outils de comparaison.

## Fonctionnalites

- Analyse fondamentale et technique d'actifs financiers
- Scoring quantitatif multi-criteres
- Comparaison de performances entre actifs
- Analyse de correlation et diversification
- Statistiques avancees (regression, distribution des rendements)
- Export de rapports

## Stack technique

- **Python** - Streamlit, Pandas, NumPy, SciPy
- **Data** - yfinance (Yahoo Finance API)
- **Visualisation** - Plotly

## Installation et lancement

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python -m streamlit run app2.py
```
